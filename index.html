Below is an example of HTML and CSS code to create a simple landing page for a coffee shop with a dark theme and modern UI: ### HTML (index.html) ```html Coffee Shop
  • Home
  • Menu
  • About

Welcome to Our Coffee Shop

Where every sip is a delight

Our Menu

  • Cappuccino
  • Latte
  • Mocha
  • Espresso

About Us

We're passionate about serving the best coffee in town.

© 2024 Coffee Shop

``` ### CSS (styles.css) ```css * { margin: 0; padding: 0; box-sizing: border-box; } body { font-family: Arial, sans-serif; background-color: #2c2c2c; color: #fff; } header { background-color: #1a1a1a; padding: 1rem; text-align: center; } header nav ul { list-style: none; } header nav ul li { display: inline-block; margin-right: 20px; } header nav a { color: #fff; text-decoration: none; } main { display: flex; flex-direction: column; align-items: center; padding: 2rem; } section { background-color: #1a1a1a; padding: 2rem; margin-bottom: 20px; border-radius: 10px; box-shadow: 0 0 10px rgba(0, 0, 0, 0.2); } h1, h2 { margin-bottom: 10px; } button { background-color: #4CAF50; color: #fff; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; } button:hover { background-color: #3e8e41; } footer { background-color: #1a1a1a; padding: 1rem; text-align: center; clear: both; } ``` This code creates a simple landing page for a coffee shop with a dark theme and modern UI. It includes a navigation menu, a hero section, a menu section, an about section, and a footer. The CSS styles the HTML elements to create a visually appealing design. Note: You'll need to create two files, `index.html` and `styles.css`, and copy the above code into them. Open the `index.html` file in a web browser to view the landing page.
